# Guía de Contribución

## Flujo de trabajo
1. Abrir un Issue.
2. Crear rama desde `main`: `tipo/descripcion-#id`.
3. Commits con referencia: `feat(docs): descripción #101`.
4. Crear PR con `Closes #101`.
5. Merge a `main` y release.

## Ramas
- main → estable
- feat/* → nuevas features
- fix/* → correcciones
- chore/* → mantenimiento
